# Product TODO

从产品角度出发，按功能域梳理 grid-table 可做的功能增强和新插件。

---

## 一、现有插件增强

### 1. filter — 列过滤

**现状**: `useFilterColumn` 仅提供 `getSelectOption()` 获取可选值列表，没有实际的过滤 UI 和逻辑。

- [ ] **筛选 UI 组件**：下拉面板（文本搜索、多选复选框），类似 Ant Design Table 的列筛选
- [ ] **多条件组合**：支持 AND/OR 逻辑组合多列过滤
- [ ] **过滤类型**：文本（包含/等于/正则）、数字（范围/大于/小于）、日期（范围）、枚举（多选）
- [ ] **过滤状态持久化**：`filterStateAtom` 存储当前过滤条件，支持外部控制和重置
- [ ] **自定义过滤函数**：`column.filterFn` 允许用户自定义过滤逻辑
- [ ] **服务端过滤模式**：仅触发回调，不在前端实际过滤数据

### 2. drag — 拖拽

**现状**: 支持列拖拽调整位置。

- [ ] **行拖拽排序**：拖拽行调整数据顺序
- [ ] **拖拽指示器**：拖拽过程中显示插入位置线和动画
- [ ] **跨表拖拽**：支持从一个表拖到另一个表
- [ ] **拖拽约束**：配置哪些行/列不可拖、拖拽范围限制

### 3. select — 行选择

**现状**: 支持 Checkbox 行选择。

- [ ] **Radio 单选模式**：`type: 'radio'` 单选
- [ ] **受控模式增强**：`selectedRowKeys` + `onChange` 完整受控 API
- [ ] **选择范围约束**：`getCheckboxProps` 控制行是否可选/禁用
- [ ] **跨页选择**：配合分页保持选中状态
- [ ] **全选性能**：大数据量下全选/反选优化

### 4. copy — 复制

**现状**: 支持框选区域复制到剪贴板。

- [ ] **格式化导出**：支持复制为 CSV / HTML / JSON 格式
- [ ] **复制带表头**：可选是否包含列标题
- [ ] **右键菜单集成**：右键选区 → 复制/剪切/粘贴
- [ ] **键盘快捷键**：Ctrl+C 复制选区，Ctrl+V 粘贴

### 5. areaSelected — 区域选择

**现状**: 鼠标拖拽选择矩形区域。

- [ ] **Shift + Click 范围选择**：点击起点后 Shift+Click 终点
- [ ] **Ctrl + Click 多区域选择**：支持不连续的多个选区
- [ ] **键盘方向键扩展选区**：Shift + Arrow 键盘扩展
- [ ] **选区高亮样式可配置**：自定义选区颜色/边框

### 6. sticky — 固定列/行

**现状**: 支持固定左/右列和顶/底行。

- [ ] **动态固定**：运行时切换列的固定状态（通过右键菜单或拖拽到边缘）
- [ ] **固定列阴影**：滚动时固定列边缘显示阴影效果（已注释掉 boxShadow 代码）
- [ ] **固定列分组**：固定列区域内支持列分组

### 7. theadContextMenu — 表头右键菜单

**现状**: 基础右键菜单。

- [ ] **内置菜单项丰富化**：排序升/降序、固定到左/右、自动列宽、隐藏列
- [ ] **自定义菜单项**：用户可注入自定义菜单项
- [ ] **子菜单**：支持多级菜单

### 8. mergeCells — 合并单元格

**现状**: 支持 tbody 和 thead 单元格合并。

- [ ] **动态合并**：基于数据自动合并相同值的相邻单元格
- [ ] **合并单元格编辑**：合并区域内的编辑交互

### 9. calcSizeByColumn — 列宽自动计算

**现状**: 测量内容自动适配列宽。

- [ ] **双击列边缘自适应**：双击表头分隔线自动适配列宽
- [ ] **列宽均分**：所有列平均分配可用宽度
- [ ] **列宽最小内容适配**：根据内容和表头标题取较大值

## 二、新插件开发

### 10. sort — 列排序 ⭐ 高优

**需求**: 表格基础功能，目前完全缺失。

- [ ] **点击表头排序**：升序/降序/取消排序循环切换
- [ ] **多列排序**：Shift+Click 添加次级排序
- [ ] **自定义排序函数**：`column.sorter` / `column.sortFn`
- [ ] **排序图标**：表头显示排序方向指示器
- [ ] **服务端排序模式**：仅触发 `onSortChange` 回调
- [ ] **sortStateAtom**：排序状态 atom 支持外部读写

### 11. columnOrder — 列顺序管理

**需求**: 运行时调整列的显示顺序。

- [ ] **拖拽调整顺序**（现有 drag 插件已部分支持，需整合为独立 API）
- [ ] **列顺序持久化**：保存/恢复列顺序
- [ ] **列顺序重置**：一键恢复默认顺序

### 12. summary / footer — 汇总行

**需求**: 表格底部/顶部汇总数据（求和、平均、计数等）。

- [ ] **固定底部汇总行**：始终可见的汇总行
- [ ] **内置聚合函数**：sum、avg、count、min、max
- [ ] **自定义汇总渲染**：`column.summary` 自定义汇总单元格
- [ ] **多行汇总**：支持多行汇总信息

### 13. columnGroup — 列分组

**需求**: 多级表头，列分组。

- [ ] **嵌套表头**：`columns` 嵌套 `children` 实现多级表头
- [ ] **分组折叠**：折叠/展开列分组
- [ ] **分组表头合并**：自动合并分组表头

### 14. rowGroup — 行分组

**需求**: 按某列值对行进行分组显示。

- [ ] **分组行头**：显示分组标题，可折叠/展开
- [ ] **分组聚合**：分组内自动聚合计算
- [ ] **多级分组**：支持嵌套分组

### 15. editCell — 单元格编辑

**需求**: `@grid-table/excel` 有编辑能力，但核心表格插件层缺失。

- [ ] **单击/双击进入编辑**：configurable 触发方式
- [ ] **编辑器类型**：input、select、datepicker、number、textarea
- [ ] **自定义编辑器**：`column.editRender` 完全自定义
- [ ] **编辑验证**：保存前校验，显示错误状态
- [ ] **编辑回调**：`onCellEdit(rowId, columnId, value)` + 批量保存

### 16. tooltip — 单元格提示

**需求**: 文字溢出时 hover 显示完整内容。

- [ ] **自动省略检测**：内容溢出时自动显示 tooltip
- [ ] **自定义 tooltip**：`column.tooltip` 自定义内容和样式
- [ ] **延迟显示**：可配置显示延迟

## 三、基础能力

### 17. 可访问性 (a11y)

**现状**: 仅 Row 上有 `role="row"`，缺乏完整 ARIA 支持。

- [ ] **ARIA 角色**：table / rowgroup / row / columnheader / cell
- [ ] **键盘导航**：Tab 在 cell 间移动，Enter 激活编辑，Escape 取消
- [ ] **屏幕阅读器**：aria-label、aria-sort、aria-selected
- [ ] **焦点管理**：活动单元格焦点环，roving tabindex

### 18. 国际化 (i18n)

**现状**: 无国际化支持，UI 文本写死。

- [ ] **locale 配置**：`<Table locale={zhCN} />`
- [ ] **内置中文/英文**：表头菜单、空状态、加载文案、筛选面板
- [ ] **自定义 locale**：用户可覆盖任意文案

### 19. 分页 (Pagination)

**现状**: 无分页支持。

- [ ] **前端分页**：内置分页器组件
- [ ] **受控分页**：`current` / `pageSize` / `onChange`
- [ ] **无限滚动**：滚动到底部自动加载下一页
- [ ] **服务端分页**：`onPageChange` 回调 + loading 状态

### 20. 导出

**现状**: 仅 copy 插件支持复制到剪贴板。

- [ ] **导出 Excel**：使用 SheetJS 导出 .xlsx
- [ ] **导出 CSV**：纯文本 CSV 下载
- [ ] **导出 PDF**：打印/PDF 导出
- [ ] **导出选区**：仅导出选中区域

### 21. 主题与样式

**现状**: CSS 变量 + BEM 命名，但无系统化主题。

- [ ] **主题系统**：亮/暗主题切换
- [ ] **CSS 变量文档**：完整的变量列表和说明
- [ ] **行条纹**：可配置的奇偶行背景色
- [ ] **紧凑/宽松模式**：`size: 'small' | 'middle' | 'large'`
- [ ] **行/列高亮**：hover 行高亮、当前行高亮

### 22. 列宽持久化与状态管理

- [ ] **列配置保存/恢复**：列宽、列顺序、列可见性、排序状态一键保存到 localStorage
- [ ] **重置默认配置**：一键恢复初始列配置

## 优先级建议

| 优先级 | 功能 | 理由 |
|--------|------|------|
| P0 | sort 排序 | 表格最基础功能，几乎所有竞品都有 |
| P1 | filter 过滤 UI | 已有数据层，缺 UI 和交互逻辑 |
| P1 | a11y 可访问性 | 合规要求，企业场景必需 |
| P1 | 行拖拽排序 | 常见交互需求 |
| P2 | summary 汇总行 | 数据表格常见功能 |
| P2 | 主题系统 | 提升开箱即用体验 |
| P2 | editCell 单元格编辑 | excel 包已有，view 层也需要轻量版 |
| P2 | columnGroup 列分组 | 复杂表头场景 |
| P3 | 导出 | 锦上添花 |
| P3 | 分页 | 虚拟滚动部分替代了分页需求 |
| P3 | i18n | 国际化覆盖 |
| P3 | tooltip | 小功能，快速实现 |
